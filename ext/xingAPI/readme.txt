▶ 설치 및 이용방법 


	1. 기존에 설치된 xingAPI를 반드시 삭제(uninstall)합니다.
	
	2. xingAPI 최신버전을 설치합니다.

	3. COM 버전 이용시에는 res 파일을 미리 다운로드 받으시면 좋습니다.
		DevCenter 로그인 후, 상단의 메뉴바 아래 [ 툴바 > Res 파일을 모두 다운로드(드럼통 이미지)] 버튼 클릭

	4. 'xingAPI 매뉴얼' 및 'DevCenter 매뉴얼'을 확인하여 사용방법을 익히시면 좋습니다.
		DevCenter 로그인 후, 상단의 [ 메뉴바 > 도움말 ] 메뉴 클릭

	5. 설치폴더 (일반적으로 C:\ETRADE\xingAPI) 아래 "샘플\엑셀VBA(COM버전용)" 폴더내의 엑셀 샘플 확인


   
▶ 특징


	1. 주식, 선물옵션, 야간선옵, 해외선물, FX마진 등 대부분의 상품을 지원합니다.

	2. 빠른 시세 및 HTS와 동일한 조회/실시간 데이터를 이용할 수 있습니다.

	3. 상세한 매뉴얼 및 프로그램 가이드를 제공합니다 ( DevCenter 로그인 - 메뉴바 - 도움말).
		=> [ 도움말 홈페이지 > 프로그램 개발 가이드] 의 내용을 따라하시면
		엑셀을 이용하여 간단한 로그인/조회/실시간 수신 프로그램을 만들 수 있습니다.
 
	4. 차트 지표데이터 및 가상거래소가 포함되어 있습니다.
		※ 차트 지표 및 가상거래소는 실서버에서만 동작합니다. (모의투자 이용 불가)


  
▶ xingAPI를 이용하거나 관련 자료를 받으시려면 


	1. 정회원 가입이 되어 있으셔야 하고,

	2. "xingAPI 사용등록/해지" 탭에서 사용등록을 하셔야 합니다.

	3. 실서버에 로그인을 하시려면 가까운 은행에서 계좌를 개설하신 후, 
       공인인증서가 있어야만 로그인 가능합니다.

	4. 모의서버에 로그인을 하시려면 당사 홈페이지 [교육센터-모의투자] 게시판을 통해 
       모의투자 참가신청을 하셔야만 합니다.

	5. 해외선물/FX마진을 이용하시려면 [ xingAPI 홈페이지 > 고객문의 ] 게시판에 문의주시면 됩니다.



▶ xingAPI에 대해서 잘 모르신다면 


	1. 증권교육센터에서 진행하는 기초 교육을 받아보시기 바랍니다.

	2. 2014/5/22(금)에 진행되었던 'API 온라인 교실' 동영상 자료가 게시되어 있습니다.
		해당 강의는 당사 교육센터에서 매월 진행하는 xingAPI COM과 DLL 교육 중 'xingAPI COM 기초' 교육을
		온라인으로 진행한 것으로 부족한 강의이지만 API를 이해하시는데에 조금이나마 도움이 되실듯 합니다.

		블로그 : http://blog.naver.com/dream_asset/220012311856
		유투브 : http://www.youtube.com/playlist?list=PLs6VjhRTJoHy6PJJjd88LQYEfOPzDZ761

	3. DevCenter 로그인 후, 상단의 [ 메뉴바 > 도움말 ] 메뉴 클릭하여 도움말을 확인합니다.

	4. 도움말 보시고, 궁금하신 점 [ xingAPI 홈페이지 > 고객문의 ] 게시판에 문의주시면 됩니다.



▶ 원하는 TR 정보가 제공되지 않는다면 

    HTS 화면번호를 기준으로 요청하시면 담당자 확인 후 관련 정보를 제공해 드립니다.




※ 기타 문의는 [ xingAPI 홈페이지 > 고객문의 ] 게시판에 문의주시면 됩니다.
   감사합니다.
